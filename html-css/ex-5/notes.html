Thought Process

Level 1

  • The layout is based on a grid system to organize content effectively across various screen sizes, ensuring responsiveness and usability.
  • The use of CSS Grid helps in defining distinct areas for the header, main content, sidebars, and footer, making the webpage flexible and easy to navigate.
  • To enhance accessibility, semantic tags like <header>, <footer>, and <article> are utilized, improving both user experience and SEO.
  • For styling, CSS variables are employed to maintain consistent colors and spacing across the webpage, ensuring easier maintenance.
  • A theme toggle was implemented to switch between light and dark modes, adding a layer of personalization.
  • Lastly, contrast between background and text colors is carefully considered for readability, and hover effects on navigation links enhance interactivity.

Level 2

  • The overall layout remains similar, but in this version, additional focus was placed on making the design more flexible and scalable.
  • The overflow property has been added to both the body and main content areas to handle excessive content gracefully, ensuring the webpage maintains usability even with more data.
  • The grid layout remains consistent, but now there's more emphasis on making the sidebars and main content scroll independently, improving the page's ability to manage content dynamically.
  • Another key difference is the use of a slightly darker primary color for the header, providing a stronger visual contrast and making the header more prominent.
  • Furthermore, this version continues to prioritize responsiveness with the inclusion of flexible grid columns, but now, with added overflow control, the design adapts even better to different screen sizes or when there's more content to display.

Level 3

  • A grid layout is used with specific areas for header, sidebars, main content, and footer.
  • Single-column layout for mobile (under 768px) and multi-column for larger screens.
  • Responsive design ensures adaptation to different screen sizes.
  • Flexbox is used in the 'main' and 'header' sections for better content alignment.
  • Dark mode is added, toggling a class on the 'body' for theme switching.
  • The layout is modular with sidebars '#sidebar1' and '#sidebar2' beside the main content.
  • Structured table design with alternating row colors for readability.
  • Fixed footer with contact information for consistent access.
